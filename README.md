# poc-bun

To install dependencies:

```bash
bun install
```

To run:

```bash
yarn start
```

To run tests:

```bash
yarn test
```

or 

Run with docker:

```bash
docker-compose up
docker run -d -p 3000:3000 poc-bun_backend
```

This project was created using `bun init` in bun v1.0.0. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
